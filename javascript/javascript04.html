01. if문

"if문"은 조건문의 한 유형으로, 특정 조건이 참(True)인 경우에만 코드 블록을 실행하도록 프로그램 흐름을 제어하는 데 사용됩니다.

{
            - true : true, "문자열", 1, 2, "1", "2", [], {} 
            - false : false, 0, null, undefined, ""(빈문자열)

            if(0){
                document.write("실행되었습니다.(true)");
            } else {
                document.write("실행되었습니다.(false)")
            }
}

"조건"은 불리언(Boolean) 값인 true 또는 false를 평가할 수 있는 표현식입니다.
만약 조건이 true로 평가되면 중괄호 {} 내부의 코드 블록이 실행됩니다. 그렇지 않으면 코드 블록은 건너뜁니다.

02. 다중 if문

다중 if 문(또는 중첩된 if 문)은 여러 개의 조건을 평가하고 다른 동작을 수행하도록 프로그램을 제어하는 데 사용됩니다. 각각의 if 문은 다른 조건을 검사하며, 첫 번째로 참으로 평가되는 조건이 만족되면 해당 if 블록 내의 코드가 실행됩니다. 이후의 if 문들은 검사하지 않습니다.

{
            const num = 100;

            if (num == 90){
                document.write("실행되었습니다.1")
            } else if(num == 95){
                document.write("실행되었습니다.2")
            } else if(num == 100){
                document.write("실행되었습니다.3")
            } else if(num == 105){
                document.write("실행되었습니다.4")
            } else {
                document.write("실행되었습니다.5")
            }
}

다중 if 문의 중요한 특징은 첫 번째 참으로 평가된 조건에서 코드가 실행되면 이후의 조건들은 더 이상 검사되지 않는다는 것입니다.
복잡한 조건을 다룰 때 유용하며, 각 조건에 따라 다른 동작을 수행해야 할 때 사용됩니다.

03. 중첩 if문

중첩 if 문은 하나의 if 문 안에 다른 if 문이나 if-else 문을 포함하는 구조를 가집니다.
이러한 중첩 구조를 사용하면 복잡한 조건을 다루거나 여러 가지 상황에 따라 다른 동작을 수행할 수 있습니다.

{
            const num = 100;

            if (num == 100){
                document.write("실행되었습니다.1");
                if (num == 100){
                    document.write("실행되었습니다.2");
                    if (num == 100){
                        document.write("실행되었습니다.3");
                    }
                }
            } else{
                document.write("실행되었습니다.4")
            }
}

다중 if 문의 중요한 특징은 첫 번째 참으로 평가된 조건에서 코드가 실행되면 이후의 조건들은 더 이상 검사되지 않는다는 것입니다.
복잡한 조건을 다룰 때 유용하며, 각 조건에 따라 다른 동작을 수행해야 할 때 사용됩니다.

04. if문 생략 & 삼항 연산자

if문은 생략할 수 있으며 삼항연산자로 식을 더 간단하게 할 수 있습니다. 

{
        const num = 100;

        if(num == 100) document.write("실행되었습니다.(true)");
        else document.write("실행되었습니다.(false)");
            
        //삼항 연산자(조건식 연산자)

        (num == 100) ? document.write("true") : document.write("false");
}

if 문을 생략하고 삼항 연산자만 사용하여 코드를 간소화할 수 있습니다.
삼항연산자: num이 100인 경우 "true"를 출력하고, 그렇지 않은 경우 "false"를 출력합니다. 삼항 연산자는 조건을 검사하고 조건이 참이면 첫 번째 표현식을 실행하고, 그렇지 않으면 두 번째 표현식을 실행합니다. 여기서 num == 100은 참이므로 "true"를 출력합니다.

05. switch문

switch 문은 JavaScript에서 사용되는 조건문 중 하나로, 여러 가지 경우(case) 중 하나를 선택하여 실행하는데 사용됩니다. switch 문은 주로 하나의 변수나 표현식의 값을 여러 경우와 비교하여 해당하는 경우에 코드 블록을 실행합니다.

{
        const num = 100;

        switch(num){
            case 90:
                document.write("실행되었습니다.(90)");
                break;
            case 95:
                document.write("실행되었습니다.(95)");
                break;
            case 100:
                document.write("실행되었습니다.(100)");
                break;
            case 105:
                document.write("실행되었습니다.(105)");
                break;
            default:
                document.write("실행되었습니다.")
        }
}

switch 문을 사용하여 num 변수의 값과 여러 경우를 비교하고 해당하는 경우에 다른 메시지를 출력합니다.

06. while문

do-while 문은 일련의 코드 블록을 실행한 다음 조건을 평가하는 반복문입니다. do 블록 내의 코드가 최소한 한 번 실행되고, 그 후에 조건을 평가하여 조건이 참인 경우 다시 do 블록을 실행합니다. 조건이 거짓이면 반복이 종료됩니다.

{
        let num = 1;
        while(num<=10){
            document.write(num);
            num++;
        }
}

let num = 1;: num이라는 변수를 선언하고 초기값을 1로 설정합니다.
while (num <= 10) {: while 키워드를 사용하여 반복문을 시작합니다. 괄호 안의 조건 (num <= 10)을 만족하는 동안 아래의 코드 블록을 반복해서 실행합니다.
document.write(num);: 현재 num 변수에 저장된 값을 웹 페이지에 출력합니다. 처음에는 1이 출력됩니다.
num++;: num 변수의 값을 1씩 증가시킵니다. 이것은 반복문에서 num 값을 업데이트하여 다음 숫자로 이동하는 역할을 합니다.
}: while 반복문의 끝을 나타냅니다. 조건 (num <= 10)이 false가 될 때까지 반복됩니다.

07. do while문

do-while 문은 일련의 코드 블록을 실행한 다음 조건을 평가하는 반복문입니다. do 블록 내의 코드가 최소한 한 번 실행되고, 그 후에 조건을 평가하여 조건이 참인 경우 다시 do 블록을 실행합니다. 조건이 거짓이면 반복이 종료됩니다.

 {
        let num = 1;
        do {
            document.write(num);
            num++;
        }while(num<=10);
}

let num = 1;: num이라는 변수를 선언하고 초기값을 1로 설정합니다. 이 변수는 반복문에서 사용될 것입니다. do {: do-while 반복문을 시작합니다. do 블록 내의 코드는 먼저 실행됩니다. document.write(num);: 현재 num 변수에 저장된 값을 웹 페이지에 출력합니다. 처음에는 1이 출력됩니다. num++;: num 변수의 값을 1씩 증가시킵니다. 이것은 반복문에서 num 값을 업데이트하여 다음 숫자로 이동하는 역할을 합니다. 따라서 다음 반복에서는 num이 2가 됩니다. }: do 블록의 끝을 나타냅니다. 이제 조건을 평가하기 전에 한 번의 반복이 완료되었습니다. while (num <= 10);: do-while 반복문의 끝을 나타냅니다. 이 부분은 do 블록을 실행한 후에 조건 (num <= 10)을 평가합니다. num이 10보다 작거나 같은 경우에는 다시 do 블록을 반복합니다. 11이 되면 조건이 거짓이 되어 반복이 종료됩니다.

08. for 문

for 문은 프로그래밍에서 가장 일반적으로 사용되는 반복문 중 하나로, 특정 횟수만큼 코드 블록을 반복 실행하는 데 사용됩니다. for 문은 초기화(initialization), 조건(condition), 반복 후 실행(statement)으로 구성됩니다.

{
        for(let i = 1; i <=10;i++){
            document.write(i)
        }
}

for (let i = 1; i <= 10; i++) {: for 키워드를 사용하여 i라는 변수를 선언하고 초기값을 1로 설정합니다.
이 변수는 반복문에서 사용될 것입니다. 세미콜론으로 구분된 세 부분으로 나눠집니다.
첫 번째 부분 (let i = 1;): 반복문이 시작될 때 한 번만 실행되는 초기화 부분입니다. 여기서는 i 변수를 1로 초기화합니다.
두 번째 부분 (i <= 10;): 반복문의 조건을 정의합니다. i가 10 이하인 동안에만 반복문이 실행됩니다.
세 번째 부분 (i++): 각 반복이 끝날 때마다 실행되는 부분으로, 여기서는 i 변수를 1씩 증가시킵니다.
{: for 반복문의 시작을 나타냅니다. 이 부분 다음에 오는 코드 블록은 반복문의 본문이 됩니다.
document.write(i): 현재 i 변수에 저장된 값을 웹 페이지에 출력합니다. i는 1부터 10까지 증가하면서 출력됩니다.
}: for 반복문의 끝을 나타냅니다. 이 부분은 반복문의 본문이 종료되는 지점입니다.
따라서 이 코드는 for 반복문을 사용하여 1부터 10까지의 숫자를 출력하며, 반복문이 실행될 때마다 i 변수가 1씩 증가하여 각 숫자가 출력됩니다.

09. break 문

break 문은 주로 반복문 (예: for, while, do-while) 내에서 사용되며, 특정 조건이 충족되면 반복문을 즉시 종료하는 역할을 합니다. break 문을 사용하면 반복문에서 빠져나와서 반복을 종료하고 다음 코드로 이동합니다.

{
        //1~50짝수
        for(i = 0;i<50;i+=2){
            document.write(i);
            if( i == 30){
                break;
            }
        }
        for(i = 0;i<50;i++){
            if(i %2 ==0){
            document.write(i)
            }
        }
}

for 반복문은 i 변수를 초기화하고, i가 50보다 작은 동안 반복을 실행합니다.
i를 2씩 증가시키므로, i는 짝수 값을 가집니다. document.write(i);는 현재 i 값을 웹 페이지에 출력합니다.
즉, 짝수를 출력하는 역할을 합니다.if (i == 30)은 조건문입니다. i가 30일 때 조건이 참이 됩니다.
break; 문은 조건이 참일 때 실행됩니다. 즉, i가 30일 때 반복문을 종료합니다.
따라서 i가 30이 되면 반복문이 종료되고, 그 이후의 짝수 출력은 일어나지 않습니다.
이 for 반복문은 i 변수를 초기화하고, i가 50보다 작은 동안 반복을 실행합니다.
if (i % 2 == 0)은 i가 짝수인지를 확인하는 조건문입니다.
% 연산자는 나머지를 계산하는 연산자이며, i를 2로 나눈 나머지가 0이면 i는 짝수입니다.
document.write(i);는 현재 i 값을 웹 페이지에 출력합니다.
이 부분은 i가 짝수인 경우에만 실행되므로, 1부터 50까지의 짝수만 출력됩니다.
이러한 방식으로 코드는 두 가지 다른 for 반복문을 사용하여 1부터 50까지의 짝수를 출력합니다.
첫 번째 반복문은 break 문을 사용하여 30까지만 출력하고 종료하며, 두 번째 반복문은 모든 짝수를 출력합니다.

10. continue문

continue 문은 반복문 (예: for, while, do-while) 내에서 사용되며, 특정 조건이 충족되면 현재 반복을 중지하고 다음 반복으로 넘어가는 역할을 합니다. continue 문을 만나면 현재 반복의 남은 부분은 실행되지 않고, 반복문의 제어가 다음 반복으로 이동합니다.

{
        //5 건너뛰고 출력
        for(i = 0;i<10;i++){
            if( i == 5){
                continue;
            }
            document.write(i)
        }
}

이 코드에서 continue 문은 i가 5일 때 실행되므로, i가 5인 경우에는 현재 반복을 중지하고 다음 반복으로 이동합니다.
따라서 출력은 0부터 9까지의 숫자 중 5만 빠져 있습니다.
