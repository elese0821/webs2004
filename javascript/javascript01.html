01. 변수 : 데이터 저장

자바스크립트(javascript)에서 변수는 데이터를 저장하는 저장소입니다.

 {
        var x = 100;            //변수 x에 숫자 100을 저장
        var y = 200;            //변수 y에 숫자 200을 저장
        var z = "javascript";   //변수 z에 문자 javascript를 저장
    
        console.log(x);
        console.log(y);
        console.log(z);
}

var는 과거에 사용되던 변수 선언 방식으로, 현재는 잘 사용하지 않습니다.

02. 변수 : 데이터 저장 + 데이터 변경

자바스크립트(javascript)에서 변수는 데이터를 변경할 수 있습니다.

{
         let x = 100;    //변수 x에 숫자 100을 저장
         let y = 200;    //변수 y에 숫자 200을 저장
         let z = "javascript"; //변수 z에 문자 javascript를 저장

         x = 300;   //변수 x에 값을 숫자 300으로 변경
         y = 400;   //변수 y에 값을 숫자 400으로 변경
         z = "jquery"; //변수 z에 값을 문자 jquery로 변경

         console.log(x)
         console.log(y)
         console.log(z)
}

let은 자바스크립트에서 변수를 선언하는 데 사용되는 키워드입니다. 블록 스코프를 가지는 변수를 선언할 때 사용합니다.

03. 변수 : 데이터 저장 + 데이터 변경 + 데이터 추가

자바스크립트(javascript)에서 변수는 데이터를 추가할 수 있습니다.

{
        let x = 100;
        let y = 200;
        let z = "javascript";

        x += 100 // x = x + 100
        y += 100 // y = y + 100
        z += "jquery" // z = z + "jquery"


        console.log(x);
        console.log(y);
        console.log(z);
}

자바스크립트에서 +=는 할당 연산자 중 하나로, 변수에 값을 더하고 그 결과를 변수에 다시 할당하는 역할을 합니다.

04. 상수 : 데이터 저장 + 데이터 변경(X)

상수는 데이터 저장이 가능하나 변경은 불가능합니다.

{
        const x = 100;
        const y = 200;
        const z = "javascript";

        x = 200; // TypeError: Assignment to constant variable.

        console.log(x);
        console.log(y);
        console.log(z);
}

const로 선언된 변수는 상수 변수를 나타내며, 한 번 값이 할당되면 그 값을 변경할 수 없습니다.

11. 배열 : 데이터 저장(여러개) : 표현방법7 : 배열 펼침연산자

자바스크립트의 배열 펼침(spread) 연산자는 배열을 펼쳐서 개별 요소로 분해하거나 여러 배열을 합치는 데 사용됩니다. 

{
        const arr1 = [100, 200, "javascript"];
        const arr2 = [300, 400, "jquery"];
        const arr3 = [...arr1, 500]

        console.log(...arr1);
        console.log(...arr1,...arr2); 
        console.log(...arr3); 
}

배열 펼침 연산자는 ... (세 개의 연속된 점) 기호로 표현되며, 다양한 상황에서 사용할 수 있습니다.

19. 객체 : 데이터 저장(키와값) : 표현방법8 : 객체 + 함수가 있는 방식

객체 + 함수 -> 객체 생성자 함수

{
        const obj = {
            a: 100,
            b: [200, 300],
            c: {a:400, b: 500, c: 600},
            d: "javascript",
            e: function(){
                console.log("jquery")
            },
            f: function(){
                console.log(obj.a)
            },
            g: function(){
                console.log(this.a)
            }
        };

        console.log(obj.b[0]);
        console.log(obj.b[1]);
        console.log(obj.c.a);
        console.log(obj.c.b);
        console.log(obj.c.c);
        console.log(obj.d);
        obj.e();
        obj.f();

22. json

JSON(JavaScript Object Notation) 데이터와 JavaScript 객체 간의 변환을 보여주는 예제입니다.

{
    //json 데이터를 객체로 변환
    const jsonString = '{"name": "webs", "age": 30}';
    const jsonObject = JSON.parse(jsonString);
    console.log(jsonObject.name);


    //객체를 json 데이터로 변환
    const person = {name: "webs", age: 30}; 
    const jsonPerson = JSON.stringify(person);
    console.log(jsonPerson);
}

1. JSON.parse() 메소드를 사용하여 JSON 문자열을 JavaScript 객체로 변환합니다.

- const jsonString = '{"name": "webs","age": "30"}';는 JSON 형식의 문자열을 정의합니다.
- JSON.parse(jsonString);는 해당 JSON 문자열을 JavaScript 객체로 변환합니다.
- console.log(jsonObject.name);은 변환된 JavaScript 객체에서 "name" 속성의 값을 출력합니다.

2. JSON.stringify() 메소드를 사용하여 JavaScript 객체를 JSON 문자열로 변환합니다.

- const person = {name: webs,age: 30};은 JavaScript 객체를 정의합니다.
- 여기서 "webs"는 문자열이어야 하므로 "webs"로 수정해야 합니다.
- JSON.stringify(person);은 JavaScript 객체 person을 JSON 형식의 문자열로 변환합니다.
- console.log(jsonObject);는 변환된 JSON 문자열을 출력합니다.

++ 연산자 : 산술 연산자

산술 연산자들은 숫자를 다루는 데 사용되며, 기본적인 산술 계산을 수행합니다.

{
        const sum = 5 + 3;  //덧셈 (+): 두 숫자를 더합니다.
        const difference = 10 - 4;  //뺄셈 (-): 한 숫자에서 다른 숫자를 뺍니다.
        const product = 2 * 6;  // 곱셈 (*): 두 숫자를 곱합니다.
        const quotient = 15 / 3;    // 나눗셈 (/): 한 숫자를 다른 숫자로 나눕니다.

        const squared = 2 ** 3; // 거듭제곱 (** 또는 Math.pow): 숫자의 거듭제곱 값을 계산합니다.

        console.log(sum);
        console.log(difference);
        console.log(product);
        console.log(quotient);
        console.log(remainder);
        console.log(squared);
}

++ 연산자 : 비교 연산자

비교 연산자는 두 값의 관계를 평가하고, 결과에 따라 true 또는 false 값을 반환합니다.

{
        var x = 5;
        var y = "5";
        console.log(x == y); // (동등 비교 연산자): 두 값이 같은지 비교합니다. 자동으로 형 변환이 발생할 수 있습니다.

        var a = 10;
        var b = "10";
        console.log(a === b); // (일치 비교 연산자): 두 값이 같은지 비교하며, 타입까지 엄격하게 비교합니다.

        var p = 7;
        var q = "7";
        console.log(p != q); // (부등 비교 연산자): 두 값이 다른지 비교합니다. 자동으로 형 변환이 발생할 수 있습니다.

        var m = 15;
        var n = "15";
        console.log(m !== n); // (불일치 비교 연산자): 두 값이 다른지 비교하며, 타입까지 엄격하게 비교합니다.

        var num1 = 8;
        var num2 = 5;
        console.log(num1 > num2);
        console.log(num1 <= num2); // > (크다), < (작다), >= (크거나 같다), <= (작거나 같다): 숫자 값을 비교할 때 사용되며, 각각 크기와 관련된 비교를 수행합니다.
}

일치 비교(===) 연산자를 사용하는 것이 값 뿐만 아니라 타입까지 엄격하게 비교하는 데 도움이 됩니다.

++ 자료형

JavaScript (JS)는 다양한 자료형을 지원하는 프로그래밍 언어입니다. 각 자료형은 특정한 종류의 데이터를 나타내며, 프로그램 내에서 다양한 목적으로 사용됩니다. 

01. 숫자(Number): 정수와 소수를 나타냅니다. 산술 연산을 수행하거나 숫자 값을 다룰 때 사용됩니다.

02. 문자열(String): 텍스트를 나타냅니다. 작은따옴표('')나 큰따옴표("")로 감싸서 표현하며, 문자열 연결(concatenation)이나 조작을 위해 사용됩니다.

03. 논리(Boolean): true 또는 false 값만을 가집니다. 조건문과 논리 연산에서 주로 사용됩니다.

04. null: '값이 없음'을 나타냅니다. 변수에 의도적으로 값이 없음을 할당할 때 사용됩니다.

05. undefined: 변수가 선언되었지만 초기화되지 않은 상태를 나타냅니다. 변수에 값이 할당되지 않았을 때 기본적으로 갖게 되는 값입니다.

06. 객체(Object): 객체는 키-값 쌍으로 구성된 집합을 나타냅니다. 객체는 다양한 데이터와 기능을 포함하며, JS에서는 거의 모든 것이 객체로 처리됩니다.

07. 배열(Array): 순차적으로 정렬된 데이터의 목록을 나타냅니다. 여러 개의 값을 하나의 변수에 저장하고 관리할 때 유용합니다.

08. 함수(Function): 함수는 코드 블록을 나타내며, 특정 작업을 수행하는 코드의 집합입니다. 재사용 가능한 코드 단위를 정의하고 호출할 때 사용됩니다.

JS에서는 이외에도 많은 자료형과 내장된 객체들이 존재합니다. 이러한 자료형들은 프로그램을 작성하고 데이터를 처리하는 데 필수적입니다. 또한 JS는 동적 타이핑(dynamic typing) 언어로, 변수의 자료형은 실행 중에 동적으로 결정되므로 유연하게 데이터를 다룰 수 있습니다.
