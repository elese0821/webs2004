21. array.prototype.join()

join()은 JavaScript에서 사용되는 배열 메서드 중 하나로, 배열의 모든 요소를 하나의 문자열로 합칠 때 사용됩니다.
이 메서드는 배열의 요소를 연결하고 구분 문자열(separator)을 사용하여 요소 사이에 삽입할 수 있습니다. 기본적으로 구분 문자열은 콤마(,)입니다.

{ 
            const fruits = ["사과", "바나나", "체리"];

            const result = fruits.join(", "); // 구분 문자열로 콤마와 공백 사용
            console.log(result); // "사과, 바나나, 체리"
}

위의 코드에서 join() 메서드를 사용하여 fruits 배열의 요소를 문자열로 합치고, 각 요소 사이에 콤마와 공백을 삽입하여 결과를 출력했습니다.
요소와 요소 사이에 다른 문자열을 사용하려면 separator 매개변수를 해당 문자열로 설정하면 됩니다.
join() 메서드는 주로 배열의 요소를 문자열로 변환하여 표시할 때 사용되며, CSV 파일을 생성하거나 URL 경로를 생성하는 데 유용합니다.

22. array.prototype.pop()

pop()은 JavaScript에서 사용되는 배열 메서드 중 하나로, 배열에서 마지막 요소를 제거하고 그 요소를 반환하는 역할을 합니다.
이 메서드는 배열을 수정하며, 배열의 길이를 하나 감소시킵니다.

{ 
            const colors = ["빨강", "파랑", "노랑"];

            const removedColor = colors.pop();
            console.log(removedColor); // "노랑"
            console.log(colors); // ["빨강", "파랑"]
}

위의 코드에서 pop() 메서드를 호출하면 colors 배열에서 마지막 요소인 "노랑"이 제거되고, 이 값을 반환하여 removedColor 변수에 저장합니다.
그 결과, removedColor에는 "노랑"이 저장되고, colors 배열은 마지막 요소가 제거된 상태인 ["빨강", "파랑"]이 됩니다.
pop() 메서드는 주로 스택(Stack) 자료 구조에서 가장 최근에 추가된 요소를 제거하는 데 사용됩니다.
