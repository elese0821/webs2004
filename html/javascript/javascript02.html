01. 변수 : 데이터 불러오기

변수를 불러오는 예제입니다.

{
        let x = 100, y = 200, z = "javascript";

        console.log(x, y, z)
}

x 변수에는 100이라는 숫자가 할당됩니다.
y 변수에는 200이라는 숫자가 할당됩니다.
z 변수에는 "javascript"라는 문자열이 할당됩니다.
그런 다음, console.log(x, y, z)를 호출하여 x, y, z의 값을 콘솔에 출력합니다.

결과 확인하기
100 200 javascript

03. 배열 : 데이터 불러오기

배열 데이터를 불러오는 예제입니다.

{
        const arr = [100, 200, "javascript"];

        console.log(arr[0], arr[1], arr[2])
}

const 키워드를 사용하여 arr이라는 상수 변수를 선언합니다.
이 변수는 배열을 저장하는데 사용됩니다.
배열인 arr에는 다음과 같은 세 가지 요소가 포함되어 있습니다:
인덱스 0에는 숫자 100이 저장됩니다.
인덱스 1에는 숫자 200이 저장됩니다.
인덱스 2에는 문자열 "javascript"가 저장됩니다.
console.log() 함수를 사용하여 배열 arr의 각 요소를 개별적으로 콘솔에 출력합니다.
각 요소는 배열의 인덱스를 통해 접근합니다. 따라서 arr[0]은 배열의 첫 번째 요소를 나타내며, arr[1]은 두 번째 요소를 나타내고, arr[2]는 세 번째 요소를 나타냅니다.

05. 배열 : 데이터 불러오기 : for()

for()문을 사용하여 데이터를 불러오는 예제입니다.

{
        const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

        for (let i = 0; i < 9; i++) {
            console.log(arr[i])
        };
}

const 키워드를 사용하여 arr이라는 상수 변수를 선언하고, 할당합니다.
이 배열에는 9개의 요소가 포함되어 있습니다.
for 반복문을 사용하여 배열 arr의 각 요소를 반복적으로 처리합니다.
초기화 부분에서 let i = 0을 통해 반복 변수 i를 0으로 초기화하고,
조건 부분에서 i < 9를 통해 반복을 진행할 조건을 설정합니다.
즉, i가 0부터 8까지 증가하면서 반복됩니다.
각 반복에서 console.log(arr[i])를 사용하여 배열 arr의 인덱스 i에 해당하는 요소를 콘솔에 출력합니다.
배열의 인덱스는 0부터 시작하므로, 첫 번째 반복에서는 arr[0]의 값을 출력하고,
두 번째 반복에서는 arr[1]의 값을 출력하고, 이런 식으로 8번 반복됩니다.
따라서 코드를 실행하면 배열 arr의 각 요소가 순서대로 콘솔에 출력됩니다.
여기에는 arr 배열의 모든 요소가 출력되지만, 배열의 길이는 9이므로 9개의 요소가 출력됩니다.

06. 배열 : 데이터 불러오기 : for Each()

forEach()메서드를 사용하여 배열 요소를 출력하는 다양한 방법을 보여주는 예제입니다.

{
        const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

        arr.forEach(function (element) {
            console.log(element);
        });

        arr.forEach(function (element, index) {
            console.log(index);
        });

        arr.forEach(function (element, index, array) {
            console.log(array);
        });

        //화살표 함수를 사용한 예시
        arr.forEach((element) => {
            console.log(element);
        }); 

        arr.forEach(element => {
            console.log(element);
        });

        arr.forEach(el => console.log(el));
}

forEach 반복문은 배열의 각 요소에 대해 주어진 함수를 실행합니다.
반환값이 없으며, 주로 배열 요소를 반복하면서 작업을 수행할 때 사용됩니다.
각각의 코드 블록에 대한 설명은 다음과 같습니다:
arr.forEach(function (element) { ... }): 배열 arr의 forEach() 메서드를 사용하여 각 요소를 반복하고, 콜백 함수를 통해 요소를 출력합니다. 이 콜백 함수는 요소 하나를 매개변수 element로 받습니다. 이 코드 블록은 배열의 요소들을 순회하면서 각 요소를 출력합니다.
arr.forEach(function (element, index) { ... }): forEach() 메서드를 사용하여 요소뿐만 아니라 요소의 인덱스도 출력합니다. 콜백 함수의 두 번째 매개변수 index를 사용하여 요소의 인덱스를 출력합니다.
arr.forEach(function (element, index, array) { ... }): forEach() 메서드를 사용하여 요소, 인덱스, 그리고 배열 자체에 접근할 수 있습니다. 이 코드에서는 배열 자체인 array를 출력합니다.
arr.forEach((element) => { ... }): 화살표 함수를 사용하여 forEach() 메서드를 호출합니다.
arr.forEach(element => { ... }): 화살표 함수를 사용하여 코드를 더 간결하게 표현합니다.
arr.forEach(el => console.log(el)): 변수 이름을 간단하게 el로 줄여서 표현합니다.

07. 배열 : 데이터 불러오기 : for of

08. 배열 : 데이터 불러오기 : for in

for...in 루프로 배열을 반복하여 요소를 출력하는 방법을 보여주는 예제입니다. 

{
        const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

        for (let element in arr) {
            console.log(element);//인덱스값
        }
        for (let element in arr) {
            console.log(arr[element]);
        }
}

for...in 루프를 사용하여 배열의 인덱스를 반복하고 각 인덱스를 출력하고,
두 번째 for...in 루프에서는 해당 인덱스를 사용하여 배열의 값을 출력합니다.
아래는 코드의 세부 내용을 설명한 것입니다:
const 키워드를 사용하여 arr이라는 상수 변수를 선언하고, 배열을 할당합니다.
이 배열에는 9개의 요소가 포함되어 있습니다. 첫 번째 for...in 루프를 사용하여 배열 arr의 각 인덱스를 반복합니다.
for...in 루프는 객체의 속성(프로퍼티)을 반복할 때 주로 사용됩니다. 이 경우, 배열의 인덱스를 가져옵니다.
let element in arr에서 element는 배열 arr의 각 인덱스를 나타냅니다.
console.log(element);를 사용하여 현재 인덱스인 element를 콘솔에 출력합니다.
두 번째 for...in 루프에서는 배열의 인덱스를 사용하여 해당 인덱스에 해당하는 값을 출력합니다.
arr[element]를 사용하여 배열 arr에서 현재 인덱스 element에 해당하는 값을 가져옵니다.
console.log(arr[element]);를 사용하여 해당 값을 콘솔에 출력합니다.

09. 배열 : 데이터 불러오기 : map()

map 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고 그 결과로 새로운 배열을 생성합니다. 원본 배열은 변경되지 않습니다.

{
        const arr = [100, 200, "javascript"];

        arr.map(function(el, index, array){
            console.log("09 : " + el + " : " + index + " : " + array);
        });

        const arrNum = arr.map((el) => {
            return el;
        });

        console.log("09 : " + arrNum)
}

1.배열 arr을 선언하고 초기값으로 [100, 200, "javascript"]를 할당합니다.
2.배열 arr에 대해 map 메서드를 호출합니다. map 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고 그 결과로 새로운 배열을 생성합니다. 주어진 함수는 요소 el, 인덱스 index, 그리고 배열 자체 array를 인수로 받습니다.
이 함수는 각 요소의 값을 로그에 출력합니다. 예를 들어, 첫 번째 호출에서는 "09 : 100 : 0 : 100,200,javascript"를 출력할 것입니다.
3.이 부분은 배열 arr을 다시 map 메서드를 사용하여 새로운 배열 arrNum에 복제하는 역할을 합니다. 화살표 함수를 사용하여 요소를 그대로 반환합니다.
4.배열 arrNum을 로그에 출력합니다. 이 배열은 [100, 200, "javascript"]와 같은 값을 가지고 있으므로 로그에는 "09 : 100,200,javascript"이 출력됩니다.

10. 배열 : 데이터 불러오기 : filter()

filter 메서드는 주어진 함수의 조건을 만족하는 배열의 모든 요소를 추출하여 새로운 배열을 반환합니다.

{
        const arr = [100, 200, "javascript"];

        arr.filter(function(el, index, array){
            console.log("10 : " + el + " : " + index + " : " + array);
        });

        const arrNum = arr.filter((el) => {
            return el > 150;
        });

        const arrNum2 = arr.filter(el => {
            return el > 150;
        }); 

        const arrNum3 = arr.filter(el =>  el > 150)

        document.write(arrNum);
        document.write(arrNum2);
        document.write(arrNum3);
}

1. 배열 arr을 선언하고 초기값으로 [100, 200, "javascript"]를 할당합니다.
2. 배열 arr에 대해 filter 메서드를 호출합니다. filter 메서드는 배열의 각 요소에 대해 주어진 함수를 호출하고 그 결과로 조건을 충족하는 새로운 배열을 생성합니다.
주어진 함수는 요소 el, 인덱스 index, 그리고 배열 자체 array를 인수로 받습니다.
이 함수는 각 요소의 값을 로그에 출력합니다. 그러나 주의해야 할 점은 filter 메서드는 새로운 배열을 생성하며, 원본 배열은 변경되지 않습니다. 따라서 이 코드에서는 filter 메서드를 사용해도 원본 배열 arr은 그대로입니다.
3. filter 메서드를 사용하여 배열 arr에서 조건을 충족하는 요소를 추출하여 새로운 배열 arrNum에 저장합니다.
조건은 각 요소 el이 150보다 큰 경우를 확인하는 것입니다
4. 이 코드는 화살표 함수를 사용하여 동일한 작업을 수행합니다. 요소 el이 150보다 큰 경우를 확인하는 조건을 가지고 있습니다.
5. 이 코드는 화살표 함수를 더 축약하여 사용합니다. 조건을 간단히 표현한 것으로, 요소 el이 150보다 큰 경우에만 참으로 간주됩니다.
6. 이 줄은 새로운 배열 arrNum을 document.write를 통해 화면에 출력하려는 시도입니다. 그러나 document.write에 배열을 전달하면 배열의 문자열 표현이 출력됩니다.
7. 이 줄은 두 번째 새로운 배열 arrNum2를 document.write를 통해 화면에 출력하려는 시도입니다.
8. 이 줄은 세 번째 새로운 배열 arrNum3을 document.write를 통해 화면에 출력하려는 시도입니다.

11. 배열 : 데이터 불러오기 : 배열 펼침 연산자

배열 펼침 연산자 (...)를 사용하여 배열을 확장하고 함수 호출 시 인수를 전달하는 예제입니다. 

{
        // 배열 펼침 연산
        const arr1 = [100, 200, 300];
        const arr2 = [400, 500, 600];

        console.log(arr1) //100, 200, 300
        console.log(...arr1) //100, 200, 300
        console.log(arr1 + arr2) //100,200, 300400,500, 600 --< concat()
        console.log(...arr1 ,400); //100, 200, 300, 400 --< push()
        console.log(...arr1.slice(0,2)); //100 200 --< slice()

        // 함수 호출시 인수 전달
        function addNum(a, b, c){
            return a+b+c;
        }

        const num = [100, 200, 300];
